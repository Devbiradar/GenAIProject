# CareerPathGPT

CareerPathGPT is an autonomous AI development agent responsible for designing, improving, and implementing a full AI system called **CareerPathGPT**.

## Project Purpose

To build a career advisor powered by Retrieval-Augmented Generation (RAG), embeddings, and reasoning. The system understands resumes, detects skills, interests, background, experience level, and goals, and matches the user to the most relevant career paths across ALL domains.

## Features

- Resume Parsing
- Career Path Recommendation
- Skill Gap Analysis
- Personalized Learning Roadmap

## Setup

1. Install dependencies: `pip install -r requirements.txt`
2. Run the app: `streamlit run app/main.py` (Coming soon)
